Data Analytics and Visualization - Tec de Monterrey - Osvaldo Alvarez

Summary: Latitude vs. X


In this project we analyzed the change in weather as one gets closer to the Equator. For doing this analysis, we obtained data from the OpenWeatherMap API and created a dataset of more than 500 cities.

Once we had the information from the API and the dataset, we used MatplotLib to plot different weather factors vs latitude. The factors we compared vs latitude were: temperature, humidity, cloudiness and wind speed. In this dashboard you can see the dataset and the different visualizations that resulted from the analysis.
